# 第1章 我们都在说谎
* 要点：
  - 创业者需要对自己撒点慌，但不能太过沉迷于幻想中。这就是我们需要数据的原因。
  - 当今创业环境可以以非常低廉的成本开发产品并做快速迭代（大模型时代尤甚）
  - 最小可行化产品：足以向市场传达你所主张的价值的最小化产品（不要求尽善尽美）。后续可以通过快速迭代搞清楚用户需求是否真实存在
  - 书籍目标：如何利用数据更快、更好地创业

# 第2章 创业的记分牌
* 数据分析的意义：在资金耗尽以前，找到正确的产品和市场。
## 数据指标设定
* 评价标准
  - 比较性强的。比如“本周用户转化率比上周高”
  - 通俗易懂。类比为“奥卡姆剃刀”
  - 是一个比率。其有点在于
    - 可操作性强，是行动的向导
    - 天生的比较性指标
    - 适用于比较各种因素间的相生和相克（正相关和负相关）
  - 会改变行为。比如指导后续的营销动作。
* 服务于同一目标，即留住用户，促进口碑传播，有效获取新用户，或者创造营收。反例就是如果指标与目标不想关，员工的商业行为就不会随之发生改变，纯纯浪费时间。
* 另一个关注点在于指标的耦合性，比如转化率与购买所需时间等
